# 🔐 Network Intrusion Detection System (NIDS)

A Rule-Based Network Intrusion Detection System developed using Python and Scapy to monitor real-time network traffic and detect suspicious activities such as DoS attacks, port scanning, and abnormal traffic patterns.

## 🚀 Features
- Real-time packet sniffing
- DoS (Denial of Service) detection
- Port scanning detection
- Suspicious port access alerts
- Time-based traffic analysis
- Logging system for detected attacks

## 🛠️ Tech Stack
- Python
- Scapy

## ⚙️ How It Works
The system captures live network packets and applies predefined rules to detect anomalies. When suspicious activity is detected, it generates alerts and logs the event for further analysis.

## 📸 Output Example
[ALERT] DoS attack suspected from 192.168.1.8

## 📂 How to Run
pip install scapy  
python detect.py
